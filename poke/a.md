# poke

父A + 母B => 子B

## 特性、道具

火焰之躯、熔岩铠甲 加速孵化

火焰之躯：鬼灯、火蜈蚣、炭山

### 不变石

子代的性格和携带不变石的亲代性格相同

母方带不变石，生出的宝可梦与母方的地区形态相同

### 红线

子代会随机遗传亲代双方的6项个体值中的5项，即6v + 6v => 5v + 1rd

### 特性

+百变怪，2个特性各50%；
+同种，70%继承母方特性

### 关于梦特

父母同种，其中有隐藏特性可以生出来

父母不同种，母方的隐藏特性可以生出来

百变怪+其他，其他的隐藏特性可以生出来

### 步骤

先确定性格 -》 优化个体 -》 刷性格
